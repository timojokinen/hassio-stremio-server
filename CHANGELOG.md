# Changelog

## 0.1.0

- Initial release
- Stremio streaming server (server.js v4.20.16) running on Node.js
- Bundled ffmpeg/ffprobe for transcoding support
- Support for amd64 and aarch64 architectures
- Configurable server.js URL for version management
- Persistent data storage across restarts
- Exposes HTTP (11470) and HTTPS (12470) streaming ports
